## List Methods (2)

Python lists come with many useful **methods** that modify the list directly.

Some commonly used list methods are:

- `.append()` – Adds an element to the end of the list.
- `.remove()` – Removes the first occurrence of a specified element.
- `.reverse()` – Reverses the order of the elements in the list.

In this exercise, you'll work with the `areas` list, which contains the areas of different parts of a house.

> [!TIP]
