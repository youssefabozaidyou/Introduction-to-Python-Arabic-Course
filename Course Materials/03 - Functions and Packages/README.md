\# Functions and Packages



\## Overview



This folder contains the Python exercises for the \*\*Functions and Packages\*\* lesson.



The exercises help learners understand how to write reusable code using functions and how to work with Python packages.



\## Contents



This folder includes:



\- Python exercise files (`.py`)

\- Exercise instruction files (`.md`)



Each Markdown file explains the exercise and provides starter code, while the Python file contains the corresponding implementation.



\## Topics Covered



\- Built-in Functions

\- Defining Functions

\- Function Arguments

\- Importing Packages

\- Using Python Modules
