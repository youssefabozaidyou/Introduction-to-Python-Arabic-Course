## Import Package

Python provides many built-in and external **packages** that contain useful tools and functions.

For example, the `math` package includes mathematical constants and functions, such as the value of **π (`pi`)**.

To calculate the circumference and area of a circle, use the following formulas:

$$
C = 2\pi r
$$

$$
A = \pi r^2
$$

Instead of typing the value of π manually, you can use `math.pi`.

> **Note:** The `**` operator is used for exponentiation in Python. For example, `3 ** 4` means 3 raised to the power of 4, which equals `81`.
