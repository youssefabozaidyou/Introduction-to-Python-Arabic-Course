## Multiple Arguments

Some Python functions accept **multiple arguments**, allowing you to customize their behavior.

In the previous exercise, you learned how to use `help()` to view a function's documentation. For example, the `sorted()` function accepts the following arguments:

- `iterable` – The collection to sort.
- `key` – An optional function used for custom sorting.
- `reverse` – An optional boolean that controls the sort order.

In this exercise, you'll use only the `iterable` and `reverse` arguments.
