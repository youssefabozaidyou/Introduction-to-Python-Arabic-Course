\# Python Files



\## Overview



This directory contains all Python source files and exercise instructions for the \*\*Introduction to Python (Arabic)\*\* course.



The content is organized by lesson, following the same order as the course videos and PDF materials. Each lesson includes multiple exercises designed to reinforce the concepts covered in that chapter.



\## Structure



```text

Python Files/

│

├── 01 - Python Basics/

├── 02 - Python Lists/

├── 03 - Functions and Packages/

└── 04 - NumPy/

```



\## Contents



Each lesson folder contains:



\- Python exercise files (`.py`)

\- Exercise instruction files (`.md`)

\- A dedicated README describing the lesson



The Markdown files include exercise descriptions, instructions, tips, and starter code, while the Python files contain the corresponding source code.
