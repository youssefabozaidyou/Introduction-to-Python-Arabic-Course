## Variable Assignment

In Python, a **variable** allows you to store a value and refer to it by a name. To create a variable named `x` with the value `10`, you use the assignment operator `=` like this:

```python
x = 10
```

You can now use the variable name `x` instead of the value `10`.

Remember, `=` in Python is used for **assignment**, not for checking equality.
