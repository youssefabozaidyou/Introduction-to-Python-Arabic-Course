\# Python Basics



\## Overview



This folder contains the Python exercises for the \*\*Python Basics\*\* lesson.



The exercises introduce the fundamental concepts of Python programming, providing hands-on practice through small coding tasks.



\## Contents



This folder includes:



\- Python exercise files (`.py`)

\- Exercise instruction files (`.md`)



Each Markdown file contains the exercise description, instructions, tips, and starter code. The corresponding Python file contains the source code for that exercise.



\## Topics Covered



\- Variables

\- Data Types

\- Basic Operations

\- Printing Output

\- Working with Values
