## Other Variable Types

In the previous exercise, you worked with the **integer** (`int`) data type.

Besides integers, Python provides several other commonly used data types:

- **`float`**: A number with a decimal point, such as `4.5`.
- **`str`**: A sequence of text enclosed in single (`' '`) or double (`" "`) quotes.
- **`bool`**: A logical value that can only be `True` or `False` (capitalization matters!).
