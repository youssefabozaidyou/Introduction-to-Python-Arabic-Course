\# NumPy



\## Overview



This folder contains the Python exercises for the \*\*NumPy\*\* lesson.



The exercises introduce NumPy arrays and demonstrate how NumPy simplifies numerical computing and data manipulation in Python.



\## Contents



This folder includes:



\- Python exercise files (`.py`)

\- Exercise instruction files (`.md`)



Each Markdown file contains the exercise description, instructions, and starter code. The Python file contains the corresponding implementation.



\## Topics Covered



\- Creating NumPy Arrays

\- Array Operations

\- Element-wise Calculations

\- Array Indexing

\- Basic NumPy Functions
