One of the biggest advantages of **NumPy arrays** is that mathematical operations are applied to **every element automatically**.

In this exercise, you'll convert a list of basketball players' weights from **pounds (lbs)** to **kilograms (kg)** using a NumPy array.

To convert pounds to kilograms, multiply each value by:

```python
0.4536
```
