## Extend a List

Besides modifying existing elements, you can also **add new elements** to a list.

One way to extend a list is by using the `+` operator:

```python
x = ["a", "b", "c", "d"]
y = x + ["e", "f"]
```

The `+` operator creates a **new list** by combining the original list with the new elements.

In this exercise, you'll add information about two new shopping items.
