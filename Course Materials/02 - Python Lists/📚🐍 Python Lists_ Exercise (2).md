## Create Lists with Different Types

Although it's not very common, a Python list can contain values of different data types, such as **strings**, **floats**, and **booleans**.

In this exercise, you'll make your `prices` list more informative by storing both the **name** of each item and its **price**.

Notice the difference:

- `"bag"` is a **string** containing the text `"bag"`.
- `bag` is a **variable** that stores the float value `35.25`.

---

## Instructions

- Complete the `prices` list by adding the missing item names as strings: `"book"`, `"shirt"`, and `"watch"` in the correct positions.
- Print the `prices` list again using `print()`.

---

> [!TIP]
> A list can store values of different data types. Strings should be enclosed in quotes, while variables should not.
