## Subsetting Lists of Lists

A Python list can contain other lists as its elements. This structure is called a **list of lists**.

To access a value inside an inner list, use two sets of square brackets:

```python
products[2][0]
```

- The first index selects the **inner list**.
- The second index selects an **element inside that inner list**.
