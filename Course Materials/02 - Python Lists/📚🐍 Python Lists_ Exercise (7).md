## Replace List Elements

You can update elements in a Python list by selecting them with their index and assigning a new value.

You can replace:

- A **single element** using its index.
- A **range of elements** using list slicing.

In this exercise, you'll update specific elements in the `prices` list.

---

## Instructions

- Update the price of the **bag** from `35.25` to `40.0` using **negative indexing**.
- Change the item name `"shoes"` to `"sneakers"` using **positive indexing**.

---

> [!TIP]
> Assign a new value directly to a list element using its index:
>
> ```python
> my_list[index] = new_value
> ```
>
> Negative indexes count from the end of the list.
