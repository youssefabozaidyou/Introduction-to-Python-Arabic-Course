\# Python Lists



\## Overview



This folder contains the Python exercises for the \*\*Python Lists\*\* lesson.



The exercises focus on working with Python lists and demonstrate how to create, access, modify, and manipulate list data.



\## Contents



This folder includes:



\- Python exercise files (`.py`)

\- Exercise instruction files (`.md`)



Each Markdown file provides the exercise description and starter code, while the Python file contains the corresponding implementation.



\## Topics Covered



\- Creating Lists

\- Accessing List Elements

\- List Slicing

\- Modifying Lists

\- List Operations
