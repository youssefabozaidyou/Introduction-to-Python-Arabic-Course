## List of Lists

As a data scientist, you'll often work with large amounts of related data. Grouping related values together makes your data easier to organize and understand.

Instead of storing item names and prices in a single flat list, you can create a **list of lists**, where each inner list contains the name of an item and its corresponding price.

Remember:

- `"book"` is a **string**.
- `book` is a **variable** that stores the value `15.50`.
